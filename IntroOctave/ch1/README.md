## 1.1 Introduction

### What is GNU Octave?

From https://www.octave.org/, you can read the slogon for GNU Octave:

**Scientific Programming Language**

- Powerful mathematics-oriented syntax with built-in 2D/3D plotting and visualization tools
- Free software, runs on GNU/Linux, macOS, BSD, and Microsoft Windows
- Drop-in compatible with many Matlab scripts

While, what is "GNU"?

A *gnu* is a type of antelope, but *GNU* is a free, Unix-like computer operating system. GNU is a recursive acronym that stands for "GNU's not Unix."
